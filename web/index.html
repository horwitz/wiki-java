Online tools

All of these have offline versions that are not subject to walltime restrictions.

Anti-spam

  • Cross-wiki linksearch: Searches multiple Wikimedia wikis for a given domain simultaneously.
  • Mass linksearch: Searches one Wikimedia wiki for multiple domains simultaneously.
  • External link checker (beta): Counts how frequently domains linked to in a given article are used in the rest of a given wiki.
  • Spam archive search: Searches archives of various spam reporting venues for a given string.

Copyright violations

  • Contribution surveyor: Lists all major edits by a given user, sorted by size.
  • Image contribution surveyor: Lists all images a user has uploaded that haven't yet been deleted on a given wiki and Wikimedia Commons.

Miscellaneous

  • Article-editor intersection: For a given set of articles, find the common editors.
  • Prefix contributions: Gets contributions of a bunch of users/IPs with a common prefix (somewhat superseded by range contributions in Special:Contributions).
  • Javadoc for Wiki.java, utilities and tools
